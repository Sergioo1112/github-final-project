# Simple Interest Calculator

## Project Overview

This project is a simple Bash-based Simple Interest Calculator designed to compute interest based on user input values. It is part of a Git and GitHub learning exercise for managing open-source projects.

## What is Simple Interest?

Simple Interest is a method of calculating interest where the interest is computed only on the original principal amount.

## Formula

The formula used in this project is:

I = (P × R × T) / 100

Where:
- P = Principal amount (initial money)
- R = Rate of interest (annual percentage)
- T = Time period (in years)
- I = Simple Interest

## How the Script Works

1. The user is prompted to enter:
   - Principal amount
   - Rate of interest
   - Time period
2. The script calculates simple interest using the formula above.
3. The result is displayed in the terminal.

## Example

Input:
- P = 1000
- R = 5
- T = 2

Output:
- Simple Interest = 100

## Usage

Run the script using:
